# Vue3 and NodeJS Small Boilerplate

# Installation
1. Clone or download this repo

2. Run this command
```
cd client
npm install
npm run serve
```

3. Open another terminal
```
cd server
npm install
npm run start:dev
```

4. Open browser, type: http://localhost:8080/
